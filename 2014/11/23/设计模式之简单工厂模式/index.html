设计模式之简单工厂模式

设计模式之简单工厂模式

标签（空格分隔）： 简单工厂模式

首先来看简单工厂模式，拿个例子说吧。军工厂制造各种装备，工厂
（Factory）,生产装备(Product),包括飞机（Airplane）,火箭(Rocket),汽车
(Car),直接看代码:

public interface Product {
public void show();

}

public class Airplane implements Product {
@Override
public void show() {
    System.out.println("我是飞机");
}

}

public class Car implements Product{
@Override
public void show() {
    System.out.println("我是汽车");
}

}

public class Rocket implements Product {
@Override
public void show() {
    System.out.println("我是火箭");
}

}

public class Factory {
public static  Product produce(String type){
    if(type.equals("rocket")){
        return new Rocket();
    }else if (type.equals("airplane")){
        return new Airplane();
    }else{
        return new Car() ;
    }
}

}

来个测试类吧：

public class Test {

public static void  main(String [] args){
    Product product = Factory.produce("rocket");
    product.show();
}

}
